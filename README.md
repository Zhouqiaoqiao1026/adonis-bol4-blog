## 技术栈

AdonisJS + node.js + bootstrap+ mongodb

## 项目运行

```
运行前你需要安装好：
node.js、mongodb、anonisjs(npm i -g @adonisjs/cli)

项目运行：
clone项目到本地
adonis serve --dev
访问：http://127.0.0.1:3333
```
